# Отключённый Node WebRTC

Klinok настраивает libp2p только с WebSocket-транспортом. Однако `@helia/libp2p`
транзитивно устанавливает `@libp2p/webrtc`, который, в свою очередь, устанавливает
нативный модуль `node-datachannel` и устаревший установщик `prebuild-install`.

Этот workspace-пакет предоставляет только интерфейс, необходимый для загрузки
`@libp2p/webrtc`, и исключает неиспользуемый нативный модуль из установки. При
попытке фактически создать Node WebRTC-соединение пакет немедленно завершит
операцию с понятной ошибкой. Если в Klinok понадобится WebRTC, эту заглушку нужно
удалить и подключить поддерживаемую реализацию транспорта.
